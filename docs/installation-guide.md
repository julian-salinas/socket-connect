# Installation guide

1. Clone the repository 

    a. Vía SSH
    ```bash
    git clone git@github.com:julian-salinas/c-connect.git
    ```

    b. Vía HTTPS
    ```bash
    git clone https://github.com/julian-salinas/c-connect.git
    ```

2. Enter the repository folder
```bash
cd c-connect
```

3. Compile and install the library
```bash
make
make install
```
